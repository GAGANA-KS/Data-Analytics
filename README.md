# Netflix Analytics Dashboard
This project analyzes Netflix movies and TV shows using Python.
It explores content trends, genres, and release patterns using
data analysis and visualization techniques.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn

## Dataset
- Netflix Movies and TV Shows dataset
- Source: Kaggle
- Contains information on title, genre, country, release year, and rating

## Features / Analysis
- Data cleaning and preprocessing
- Analysis of movies vs TV shows
- Genre-wise content distribution
- Year-wise content growth analysis
- Visualization of trends using charts
